Accessibility Statement

WCAG 2.1 Level AA Compliant

Commitment to Accessibility

This portfolio website is designed to be accessible to all users, including those with disabilities. It meets WCAG 2.1 Level AA standards.

Features

  • Semantic HTML5: Proper document structure
  • ARIA Labels: Screen reader support
  • Keyboard Navigation: Full Tab key support
  • Color Contrast: 4.5:1 ratio (WCAG AA)
  • Responsive: Works on all devices
  • Focus Indicators: Visible on all elements

Testing

This site has been tested with automated tools and manual keyboard navigation.

If you encounter any accessibility issues, please contact me.
